## Notebook: Control-Volume Mass Balance

This notebook introduces mass conservation for a control volume using
the Reynolds Transport Theorem.

It includes:

- Basic functions for cross-sectional area and mass flow rate
- A steady incompressible nozzle
- A steady mixing chamber with multiple inlets
- An unsteady tank-filling problem
- Control-volume diagrams
- Mass-balance residual checks
- Liquid-height visualization and linear-trend verification
